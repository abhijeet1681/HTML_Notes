HTML Colors

Standard Color - The color code goes by the name itself.
RGB Color Code - The rgb color code value should be between 0 to 255.
Hex Color Code - The hex code value will be between 0 to 9 and a to f.
HSL(Hue, Saturation, Light) Color Code - The h will be from 0 to 360 and s,l will be from 0 to 100%.
